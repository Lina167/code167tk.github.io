HTML Basic

ក្នុង​ជំពូក​នេះ យើង​នឹង​បង្ហាញ​ឧទាហរណ៍​មូលដ្ឋាន​ HTML មួយ​ចំនួន។កុំបារម្ភប្រសិនបើយើងប្រើស្លាកដែលអ្នកមិនទាន់បានសិក្សា។
❂ ឯកសារ HTML
HTML ទាំងអស់ត្រូវតែចាប់ផ្តើមជាមួយនឹងការប្រកាសប្រភេទឯកសារ៖ <!DOCTYPE html>.
ឯកសារ HTML ខ្លួនវាចាប់ផ្តើមដោយ <html> និងបញ្ចប់ដោយ </html>
ផ្នែកដែលអាចមើលឃើញនៃឯកសារ HTML គឺនៅចន្លោះ <body> និង </body>.
❂ ឧទាហរណ៍

<!DOCTYPE html>
 <html>
 <head>
 <title> Page Title </title>
 </head>
 <body>

  <h1> Hello ReaCode </h1>
  <p> HTML ជាអក្សរកាត់នៃពាក្យពេញ HyperText Markup Language មានន័យថាភាសាដែលប្រើលើសពីការតុបតែងអក្សរ។ វាត្រូវបានបង្កើតក្នុងគោលបំណងផ្លាស់ពត៌មាននៅលើ internet។ </p>

 </body>
 <html>


❂ The <!DOCTYPE> Declaration
សេចក្តី <!DOCTYPE>ប្រកាសតំណាងឱ្យប្រភេទឯកសារ និងជួយកម្មវិធីរុករកឱ្យបង្ហាញគេហទំព័របានត្រឹមត្រូវ។វាត្រូវតែបង្ហាញតែម្តងគត់ នៅផ្នែកខាងលើនៃទំព័រ (មុនស្លាក HTML ណាមួយ)។
សេចក្តី <!DOCTYPE> ប្រកាសមិនប្រកាន់អក្សរតូចធំទេ។
សេចក្តី <!DOCTYPE> ប្រកាសសម្រាប់ HTML5 គឺ៖
  <!DOCTYPE html> 

❂ ចំណងជើង HTML
ក្បាល HTML ត្រូវបានកំណត់ជាមួយនឹង < h1 > to <h6> tag
< h1 > កំណត់ចំណងជើងសំខាន់បំផុត
<h6> កំណត់​ក្បាល​សំខាន់​តិច​បំផុត៖

❂ HTML Paragraphs
Paragraphs ត្រូវបានកំណត់ជាមួយ < p >

< p > This is a paragraph. < /p >
< p > This is another paragraph. < /p >


❂ HTML Links
តំណភ្ជាប់ HTML ត្រូវបានកំណត់ជាមួយ < a > tag.
ទិសដៅរបស់តំណត្រូវបានបញ្ជាក់នៅក្នុង hrefគុណលក្ខណៈត្រូវបានប្រើដើម្បីផ្តល់ព័ត៌មានបន្ថែមអំពីធាតុ HTML អ្នកនឹងស្វែងយល់បន្ថែមអំពីគុណលក្ខណៈនៅក្នុងជំពូកបន្ទាប់។

< a href="https://lina167.github. io/reancodfree.github.io"> This is a link < /a >

❂ HTML Images
រូបភាព HTML ត្រូវបានកំណត់ជាមួយ < img > tag.
ឯកសារប្រភព (src) អត្ថបទជំនួស width និង height ត្រូវបានផ្តល់ជាគុណលក្ខណៈ៖

< img src="reancode.png" alt="reancode.com" width="104" height="142" >
