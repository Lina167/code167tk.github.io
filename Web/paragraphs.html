HTML Paragraphs

កថាខណ្ឌតែងតែចាប់ផ្តើមនៅលើបន្ទាត់ថ្មី ហើយជាធម្មតាជាប្លុកនៃអត្ថបទ។
❂ កថាខណ្ឌ HTML
កថាខណ្ឌតែងតែចាប់ផ្តើមនៅលើបន្ទាត់ថ្មី ហើយកម្មវិធីរុករកបន្ថែមចន្លោះពណ៌សមួយចំនួន (រឹម) មុន និងក្រោយកថាខណ្ឌដោយស្វ័យប្រវត្តិ។

<p>This is a paragraphs</p>
<p>This is another paragraph.</p>

❂ ការបង្ហាញ HTML
អ្នក​មិន​អាច​ប្រាកដ​ថា​របៀប​ដែល HTML នឹង​ត្រូវ​បាន​បង្ហាញ​។ អេក្រង់ធំ ឬតូច និងបង្អួចដែលផ្លាស់ប្តូរទំហំនឹងបង្កើតលទ្ធផលខុសៗគ្នា។ ជាមួយ HTML អ្នកមិនអាចផ្លាស់ប្តូរការបង្ហាញដោយបន្ថែមចន្លោះបន្ថែម ឬបន្ទាត់បន្ថែមនៅក្នុងកូដ HTML របស់អ្នកបានទេ។ កម្មវិធីរុករកតាមអ៊ីនធឺណិតនឹងដកចន្លោះ និងបន្ទាត់បន្ថែមដោយស្វ័យប្រវត្តិ នៅពេលដែលទំព័រត្រូវបានបង្ហាញ៖

<p>This paragraph contains a lot of lines in the source code, but the browser ignores it.</p>
<p>This paragraph contains a lot of spaces in the source code, but the browser ignores it.</p>
សាកល្បងដោយខ្លួនឯង

❂ ច្បាប់ផ្តេក HTML
ស្លាក <hr> កំណត់ការបំបែកប្រធានបទនៅក្នុងទំព័រ HTML ហើយជាញឹកញាប់បំផុតត្រូវបានបង្ហាញជាក្បួនផ្ដេក។
ស្លាក <hr> ត្រូវបានប្រើដើម្បីបំបែកមាតិកា (ឬកំណត់ការផ្លាស់ប្តូរ) នៅក្នុងទំព័រ HTML

<h1>This is heading 1</h1>
<p>This is some text. </p>
<hr>
<h2>This is heading 2</h2>
<p>This is some text. </p>
<hr>
សាកល្បងដោយខ្លួនឯង

<hr> គឺជាស្លាកទទេដែលមានន័យថាវាមិនមានស្លាកបញ្ចប់។

❂ ការចុះបន្ទាត់ HTML
​ធាតុ HTML <br> កំណត់ការចុះបន្ទាត់។ ប្រើ <br> ប្រសិនបើអ្នកចង់ចុះបន្ទាត់ (បន្ទាត់ថ្មី) ដោយមិនចាំបាច់ចាប់ផ្តើមកថាខណ្ឌថ្មី៖

<p>This is <br> a paragraph <br> with line breaks. </p>
សាកល្បងដោយខ្លួនឯង

<br> គឺជាស្លាកទទេដែលមានន័យថាវាមិនមានស្លាកបញ្ចប់។

❂ ដំណោះស្រាយ HTML <pre> Element
ធាតុ <pre> Element កំណត់អត្ថបទដែលបានធ្វើទ្រង់ទ្រាយជាមុន។ អត្ថបទនៅខាងក្នុង <pre> ធាតុមួយត្រូវបានបង្ហាញជាពុម្ពអក្សរដែលមានទទឹងថេរ (ជាធម្មតាអ្នកនាំសំបុត្រ) ហើយវារក្សាទាំងចន្លោះ និងការបំបែកបន្ទាត់៖

<pre>
My Bonnie lies over the ocean.
My Bonnie lies over the sea.
My Bonnie lies over the ocean.
Oh, bring back my Bonnie to me.
</pre>
សាកល្បងដោយខ្លួនឯង
