❂ រៀន HTML ប្រើ Notepad ឬ TextEdit
  ទំព័របណ្ដាញអាចត្រូវបានបង្កើតនិងកែប្រែដោយប្រើកម្មវិធី HTML editors ។ទោះជាយ៉ាង ណាក៏ដោយសម្រាប់ការរៀនHTMLយើងសូមផ្តល់អនុសាសន៍ឲ្យអ្នកនិពន្ធអត្ថបទសាមញ្ញដូចជាNotepad (PC) ឬ TextEdit (Mac)។យើងជឿថាការប្រើកម្មវិធីនិពន្ធអត្ថបទសាមញ្ញគឺជាវិធីដ៏ល្អមួយដើម្បីរៀន HTML ។សូមធ្វើតាមជំហានខាងក្រោមដើម្បីបង្កើតទំព័របណ្ដាញដំបូងរបស់អ្នកជាមួយ Notepad ឬ TextEdit ។


❂ ជំហានទី១: បើក Notepad (PC)
Windows 8 or later:
Open the Start Screen (the window symbol at the bottom left on your screen). Type Notepad.


❂ ជំហានទី១: បើក TextEdit (Mac)
ការផ្លាស់ប្តូរចំណង់ចំណូលចិត្តមួយចំនួនដើម្បីទទួលបានកម្មវិធីទៅ រក្សាទុក ឯកសារ ឲ្យ បាន ត្រឹមត្រូវ ។ នៅក្នុង Preferences > Format > ជ្រើសរើស "អត្ថបទធម្មតា"
បន្ទាប់មកក្រោម"បើកនិងរក្សាទុក"សូមពិនិត្យមើលប្រអប់ដែលនិយាយថា"បង្ហាញ ឯកសារ HTML ជាកូដ HTMLជំនួសឲ្យការធ្វើទ្រង់ទ្រាយអត្ថបទ"។
បន្ទាប់មកបើកឯកសារថ្មីមួយដើម្បីដាក់លេខកូដ។


❂ ជំហានទី 2: សរសេរ HTML មួយចំនួន
សរសេរ ឬចម្លងកូដ HTML ខាងក្រោមទៅក្នុង Notepad៖


❂ ជំហានទី 3: រក្សាទុកទំព័រ HTML
សរសេរ ឬចម្លងកូដ HTML ខាងក្រោមទៅក្នុង Notepad៖

រក្សាទុកឯកសារនៅលើកុំព្យូទ័ររបស់អ្នក។ ជ្រើសរើស ឯកសារ> រក្សាទុកដូច នៅក្នុងម៉ឺនុយ Notepad ។
ដាក់ឈ្មោះឯកសារ "index.htm" ហើយកំណត់ការអ៊ិនកូដទៅជា UTF-8 (ដែលជាការអ៊ិនកូដដែលពេញចិត្តសម្រាប់ឯកសារ HTML)។
