ការណែនាំ HTML

តើ HTML ជាអ្វី?
 HTML ជាអក្សរកាត់នៃពាក្យពេញ HyperText Markup Language មានន័យថា ភាសាដែល ប្រើលើសពីការតុបតែងអក្សរ។ វាត្រូវបានបង្កើតក្នុងគោលបំណងផ្លាស់ពត៌មាននៅលើ internet។
❂ ឧទាហរណ៍

<!DOCTYPE html>
 <html>
 <head>
 <title> Page Title </title>
 </head>
 <body>

  <h1> Hello ReaCode </h1>
  <p> HTML ជាអក្សរកាត់នៃពាក្យពេញ HyperText Markup Language មានន័យថាភាសាដែលប្រើលើសពីការតុបតែងអក្សរ។ វាត្រូវបានបង្កើតក្នុងគោលបំណងផ្លាស់ពត៌មាននៅលើ internet។ </p>

 </body>
 <html>


❂ ពន្យល់
  ❖ <!DOCTYPE html> ប្រកាសកំណត់ថាឯកសារនេះគឺជាឯកសារ HTML5
  ❖ <html> គឺជាធាតុដើមនៃទំព័រ HTML
  ❖ <head> មានព័ត៌មាន Meta អំពីទំព័រ HTML
  ❖ <title> បញ្ជាក់ចំណងជើងសម្រាប់ទំព័រ HTML (ដែលត្រូវបានបង្ហាញនៅក្នុងរបារចំណងជើងរបស់កម្មវិធីរុករក ឬនៅក្នុងផ្ទាំងរបស់ទំព័រ)
  ❖ <body> កំណត់តួឯកសារ និងជាធុងសម្រាប់មាតិកាដែលមើលឃើញទាំងអស់ ដូចជា ចំណងជើង កថាខណ្ឌ រូបភាព link តារាង បញ្ជី។ល។
  ❖ <h1> កំណត់ក្បាលអក្សរ
  ❖ <p> element defines a paragraph

តើ HTML Element ជាអ្វី?
 HTML Element ត្រូវបានកំណត់ដោយ start tag មាតិកាមួយចំនួននិងបញ្ចប់ដោយ end tag។
 <tagname> Content goes here... </tagname>
Element Content ជាអក្សរនៅក្នុងចន្លោះ start tag និង end tag ។

Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
</br> none none

ចំណាំ៖ HTML Element ខ្លះមិនមានខ្លឹមសារទេ (ដូចជាធាតុ </br>)។ ធាតុទាំងនេះត្រូវបានគេហៅថាធាតុទទេ។ ធាតុទទេមិនមានស្លាកបញ្ចប់ទេ!


❂ Web Browsers
គោលបំណងនៃកម្មវិធីរុករកបណ្ដាញ (Chrome, Edge, Firefox, Safari) គឺដើម្បីអានឯកសារ HTML និងបង្ហាញពួកវាឱ្យបានត្រឹមត្រូវ។
កម្មវិធីរុករកតាមអ៊ីនធឺណិតមិនបង្ហាញស្លាក HTML ទេ ប៉ុន្តែប្រើពួកវាដើម្បីកំណត់ពីរបៀបបង្ហាញឯកសារ៖

❂ រចនាសម្ព័ន្ធទំព័រ HTML
ខាង​ក្រោម​នេះ​គឺ​ជា​ទិដ្ឋភាព​នៃ​រចនា​សម្ព័ន្ធ​ទំព័រ HTML៖

ចំណាំ៖ ខ្លឹមសារនៅក្នុងផ្នែក <body>(ផ្ទៃពណ៌សខាងលើ) នឹងត្រូវបានបង្ហាញនៅក្នុងកម្មវិធីរុករក។ ខ្លឹមសារនៅក្នុងធាតុ <title> នឹងត្រូវបានបង្ហាញនៅក្នុងរបារចំណងជើងរបស់កម្មវិធីរុករកតាមអ៊ីនធឺណិត ឬនៅក្នុងផ្ទាំងរបស់ទំព័រ។
