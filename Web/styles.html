HTML Styles

HTML style ត្រូវបានប្រើដើម្បីបន្ថែមរចនាប័ទ្មទៅធាតុមួយ ដូចជាពណ៌ ពុម្ពអក្សរ ទំហំ និងច្រើនទៀត។

I'm red

I'm blue

I'm big

សាកល្បងដោយខ្លួនឯង

❂ គុណលក្ខណៈរចនាប័ទ្ម HTML
ការកំណត់រចនាប័ទ្មនៃធាតុ HTML អាចត្រូវបានធ្វើដោយ style គុណលក្ខណៈ។គុណលក្ខណៈ HTML style មានវាក្យសម្ព័ន្ធដូចខាងក្រោមៈ

<tagname style="property:value;" >

អ្នកនឹងស្វែងយល់បន្ថែមអំពី CSS នៅពេលក្រោយនៅក្នុងមេរៀននេះ។

❂ ពណ៌ផ្ទៃខាងក្រោយ
លក្ខណសម្បត្តិ CSS background-color កំណត់ពណ៌ផ្ទៃខាងក្រោយសម្រាប់ធាតុ HTML ។
កំណត់ពណ៌ផ្ទៃខាងក្រោយសម្រាប់ទំព័រទៅជាម្សៅពណ៌ខៀវ៖

<body style="background-color:powderblue;">
<h1> This is a heading </h1>
<p> This is a paragraph </p>
សាកល្បងដោយខ្លួនឯង
