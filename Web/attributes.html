HTML Attributes

គុណលក្ខណៈ HTML ផ្តល់ព័ត៌មានបន្ថែមអំពីធាតុ HTML ។


❂ គុណលក្ខណៈ HTML
⇝ ធាតុ HTML ទាំងអស់អាចមាន គុណលក្ខណៈ
⇝ គុណលក្ខណៈផ្តល់ ព័ត៌មានបន្ថែម អំពីធាតុ
⇝ គុណលក្ខណៈត្រូវបានបញ្ជាក់ជានិច្ចនៅក្នុង ស្លាកចាប់ផ្តើម
⇝ គុណលក្ខណៈជាធម្មតាមានជាគូឈ្មោះ/តម្លៃដូចជា៖ name="value"


❂ គុណលក្ខណៈ href
ស្លាក <a> កំណត់តំណខ្ពស់មួយ។ គុណលក្ខណៈ hrefបញ្ជាក់ URL នៃទំព័រដែលតំណទៅ៖

❂ ឧទាហរណ៍

<a  href="https://lina167.github.io/reancode167.github.io/"> Wellcome To ReanCode </a>


❂ គុណលក្ខណៈ src
ស្លាក <img> ត្រូវបានប្រើដើម្បីបង្កប់រូបភាពក្នុងទំព័រ HTML ។ គុណលក្ខណៈ srcបញ្ជាក់ផ្លូវទៅកាន់រូបភាពដែលត្រូវបង្ហាញ៖
❂ ឧទាហរណ៍

<img src="img_girl.jpg">

មានវិធីពីរយ៉ាងក្នុងការបញ្ជាក់ URL ក្នុង src គុណលក្ខណៈ៖
1. URL ដាច់ខាត - តំណភ្ជាប់ទៅរូបភាពខាងក្រៅដែលបង្ហោះនៅលើគេហទំព័រផ្សេងទៀត។ ឧទាហរណ៍៖ src="https://lina167.github.io/reancode167.github.io/images/img_girl.jpg" ។
ចំណាំ៖ រូបភាពខាងក្រៅអាចស្ថិតនៅក្រោមការរក្សាសិទ្ធិ។ ប្រសិនបើអ្នកមិនទទួលបានការអនុញ្ញាតឱ្យប្រើវាទេ អ្នកអាចនឹងបំពានច្បាប់រក្សាសិទ្ធិ។ លើសពីនេះ អ្នកមិនអាចគ្រប់គ្រងរូបភាពខាងក្រៅបានទេ។ ភ្លាមៗនោះវាអាចត្រូវបានលុបចោល ឬផ្លាស់ប្តូរ។
2. URL ដែលទាក់ទង - តំណភ្ជាប់ទៅរូបភាពដែលបង្ហោះនៅក្នុងគេហទំព័រ។ នៅទីនេះ URL មិនរួមបញ្ចូលឈ្មោះដែន។ ប្រសិនបើ URL ចាប់ផ្តើមដោយគ្មានសញ្ញា វានឹងទាក់ទងទៅនឹងទំព័របច្ចុប្បន្ន។ ឧទាហរណ៍៖ src="img_girl.jpg" ។ ប្រសិនបើ URL ចាប់ផ្តើមដោយសញ្ញា វានឹងទាក់ទងទៅនឹងដែន។ ឧទាហរណ៍៖ src="/images/img_girl.jpg" ។
គន្លឹះ៖ វាស្ទើរតែតែងតែល្អបំផុតក្នុងការប្រើ URL ដែលទាក់ទង។ ពួកគេនឹងមិនខូចទេប្រសិនបើអ្នកប្តូរដែន។

❂ គុណលក្ខណៈទទឹងនិងកំពស់
ស្លាក <img> ក៏គួរតែមាន width និង height គុណលក្ខណៈផងដែរ ដែលបញ្ជាក់ទទឹង និងកម្ពស់នៃរូបភាព (គិតជា pixels)៖
❂ ឧទាហរណ៍

<img src="img_girl.jpg" width="500" height="600">

❂ គុណលក្ខណៈ alt
គុណលក្ខណៈ ដែលត្រូវការ alt សម្រាប់ <img> ស្លាកបញ្ជាក់អត្ថបទជំនួសសម្រាប់រូបភាព ប្រសិនបើរូបភាពសម្រាប់ហេតុផលមួយចំនួនមិនអាចបង្ហាញបាន។ នេះអាចបណ្តាលមកពីការតភ្ជាប់យឺត ឬកំហុសនៅក្នុង srcគុណលក្ខណៈ ឬប្រសិនបើអ្នកប្រើប្រើកម្មវិធីអានអេក្រង់។
❂ ឧទាហរណ៍

<img  src="img_girl.jpg" alt="Girl with a jacket">

❂ style Attribute
គុណលក្ខណៈ style ត្រូវបានប្រើដើម្បីបន្ថែមរចនាប័ទ្មទៅធាតុមួយ ដូចជាពណ៌ ពុម្ពអក្សរ ទំហំ និងច្រើនទៀត។
❂ ឧទាហរណ៍

<p style="color:red;"> This is a red paragraph. <p>

❂ គុណលក្ខណៈភាសា
អ្នកគួរតែបញ្ចូល langគុណលក្ខណៈនៅក្នុង <html> ស្លាកជានិច្ច ដើម្បីប្រកាសភាសានៃគេហទំព័រ។ នេះមានន័យដើម្បីជួយដល់ម៉ាស៊ីនស្វែងរក និងកម្មវិធីរុករក។
ឧទាហរណ៍ខាងក្រោមបញ្ជាក់ភាសាអង់គ្លេសជាភាសា៖

<!DOCTYPE html>
 <html lang="en">
 <body>
  -----
 </body>
 <html>

លេខកូដប្រទេសក៏អាចត្រូវបានបន្ថែមទៅលេខកូដភាសានៅក្នុង lang គុណលក្ខណៈផងដែរ។ ដូច្នេះ តួអក្សរពីរដំបូងកំណត់ភាសានៃទំព័រ HTML ហើយតួអក្សរពីរចុងក្រោយកំណត់ប្រទេស។
ឧទាហរណ៍ខាងក្រោមបញ្ជាក់ភាសាអង់គ្លេសជាភាសា និងសហរដ្ឋអាមេរិកជាប្រទេស៖

<!DOCTYPE html>
 <html lang="en-US">
 <body>
  -----
 </body>
 <html>

❂ គុណលក្ខណៈចំណងជើង
គុណលក្ខណៈ titleកំណត់ព័ត៌មានបន្ថែមមួយចំនួនអំពីធាតុមួយ។
តម្លៃនៃគុណលក្ខណៈចំណងជើងនឹងត្រូវបានបង្ហាញជាព័ត៌មានជំនួយ នៅពេលអ្នកដាក់កណ្ដុរលើធាតុ៖
❂ ឧទាហរណ៍

<p title="I'm a tooltip"> This is a red paragraph. <p>

❂ ជំពូកសង្ខេប
⇝ ធាតុ HTML ទាំងអស់អាចមាន គុណលក្ខណៈ
⇝ គុណលក្ខណៈ href នៃ <a> ការបញ្ជាក់ URL នៃទំព័រដែលតំណទៅ
⇝ គុណលក្ខណៈ src នៃ <img> ការកំណត់ផ្លូវទៅកាន់រូបភាពដែលត្រូវបង្ហាញ
⇝ និង width គុណលក្ខណៈ height នៃ <img> ការផ្តល់ព័ត៌មានទំហំសម្រាប់រូបភាព
⇝ គុណលក្ខណៈ alt នៃ <img> ផ្តល់នូវអត្ថបទជំនួសសម្រាប់រូបភាពមួយ។
⇝ គុណលក្ខណៈ style ត្រូវបានប្រើដើម្បីបន្ថែមរចនាប័ទ្មទៅធាតុមួយ ដូចជាពណ៌ ពុម្ពអក្សរ ទំហំ និងច្រើនទៀត
⇝ គុណលក្ខណៈ lang នៃ <html> ស្លាកប្រកាសភាសានៃគេហទំព័រ
⇝ គុណលក្ខណៈ title កំណត់ព័ត៌មានបន្ថែមមួយចំនួនអំពីធាតុមួយ។
